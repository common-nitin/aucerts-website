# Aucerts Website

This is a static HTML/CSS/JavaScript website for Aucerts Pty Ltd.

## Main files

- `index.html` - Home page
- `about.html` - About page
- `services.html` - Services page
- `vehicle-approvals.html` - Vehicle approvals page
- `component-approvals.html` - Component approvals page
- `engineering.html` - Engineering / CAD / FEA page
- `international-homologation.html` - International homologation page
- `regulations.html` - ADR and regulations page
- `blog.html` - Blog page
- `contact.html` - Contact page
- `assets/css/style.css` - All website styling
- `assets/js/main.js` - Mobile menu script
- `assets/images/aucerts-logo.png` - Logo
- `CNAME` - Custom domain file for GitHub Pages
- `.nojekyll` - Keeps GitHub Pages simple for static HTML

## How to preview locally

Open `index.html` in your browser.

## How to upload to GitHub

From this folder:

```bash
git init
git add .
git commit -m "Initial Aucerts website"
git branch -M main
git remote add origin https://github.com/common.nitin/common.nitin.github.io.git
git push -u origin main
```

Then enable GitHub Pages from repository settings.
